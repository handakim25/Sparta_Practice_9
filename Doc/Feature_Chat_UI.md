# Chat UI

## 구현 목적

유저에게 채팅을 칠 수 있는 UI와 채팅을 받아서 확인할 수 있는 UI를 작성한다.

## 필요한 기능

- 사용자로부터 입력을 받을 텍스트박스 UI
- 사용자의 입력을 받아서 출력할 리스트 박스

### 리스트 박스 기능

- [x] 입력되면 가장 밑으로 내리는 기능
- [x] 끝에 존재할 때만 가장 밑으로 스크롤 기능
- [x] 2줄 이상의 텍스트 출력

### 입력 박스 기능

- [X] 최대 문자 길이 설정

### 스크롤 박스

- [x] 배경 이미지 추가

## 추가 기능

- 너무 오래된 텍스트를 제거하는 기능
- 오브젝트 풀링
